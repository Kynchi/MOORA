# Deskripsi Proyek
Sistem Pendukung Evaluasi Kinerja Pegawai â€“ Metode MOORA adalah aplikasi berbasis web yang dirancang untuk membantu organisasi dalam menilai dan membandingkan kinerja pegawai secara objektif dan terstruktur. Sistem ini mengimplementasikan metode MOORA (Multi-Objective Optimization on the Basis of Ratio Analysis), sebuah pendekatan pengambilan keputusan multikriteria yang kuat, efisien, dan transparan.

Dengan metode MOORA, sistem ini mampu menangani berbagai kriteria penilaian baik yang bersifat benefit maupun cost, lalu mengolah data kuantitatif dari performa pegawai menjadi nilai akhir yang terstandardisasi. Hasil evaluasi ditampilkan dalam bentuk peringkat, grafik, dan tabel komparatif yang memudahkan manajer atau pimpinan untuk mengambil keputusan berbasis data.

# Fitur Utama
* Unggah Data Excel: Pengguna dapat mengunggah data penilaian karyawan secara massal melalui file Excel.
* Normalisasi Otomatis: Sistem secara otomatis melakukan normalisasi dan klasifikasi kriteria benefit dan cost.
* Perhitungan MOORA: Implementasi penuh dari perhitungan MOORA, termasuk nilai terbobot dan nilai optimal.
* Visualisasi Hasil: Hasil evaluasi disajikan dalam bentuk tabel peringkat dan visualisasi untuk mempermudah pemahaman.
